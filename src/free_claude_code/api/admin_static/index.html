Proxy Pool

Fetches the URL and adds each proxy line. Lines with a scheme (http/https/socks5) are kept as-is; bare host:port lines become HTTP proxies. Duplicates are skipped.

Proxies in this pool are automatically used by all providers (including built-in ones like NVIDIA NIM). When the pool is empty, providers fall back to their individual proxy settings. Use "Test" to verify connectivity before they are used.
